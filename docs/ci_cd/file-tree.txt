Quick Command Reference
=======================

Local Testing:
  cd firmware/v2/test/tools && ./test_tools.sh

Run Benchmark:
  cd v2
  pio test -e native_test -f test_pipeline_benchmark --verbose > benchmark.log

Parse Results:
  python test/tools/parse_benchmark_serial.py benchmark.log --platformio --format summary

Check Regression:
  python test/tools/detect_regressions.py \
    test/baseline/benchmark_baseline.json \
    <(python test/tools/parse_benchmark_serial.py benchmark.log --platformio)
