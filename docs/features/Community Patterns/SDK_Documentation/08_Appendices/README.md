## üéØ Getting Help Effectively

### **Before Asking for Help**
1. **Check Documentation** ‚Üí Search this documentation suite
2. **Review Troubleshooting** ‚Üí Check common issues
3. **Search Community** ‚Üí Look for similar questions
4. **Test Systematically** ‚Üí Isolate the problem

### **When Asking for Help**
1. **Describe the Problem** ‚Üí Clear, specific description
2. **Provide Context** ‚Üí What you were trying to do
3. **Include Details** ‚Üí Error messages, code snippets, logs
4. **Show What You Tried** ‚Üí Steps you've already taken
5. **Specify Environment** ‚Üí Hardware, software versions

### **Providing Good Bug Reports**
```
Title: Clear, specific summary of the issue

Environment:
- K1 Lightwave Firmware: vX.X.X
- SDK Version: vX.X.X
- Development Platform: Windows/Mac/Linux
- IDE: VS Code/Arduino IDE/etc.

Steps to Reproduce:
1. First step
2. Second step
3. Third step

Expected Behavior:
What should have happened

Actual Behavior:
What actually happened

Additional Information:
- Error messages (exact text)
- Code snippets (minimal example)
- Logs or console output
- Screenshots/videos if relevant
```
