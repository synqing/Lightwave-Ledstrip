Transition Specification
Collapse trigger Tap anywhere on the header row (full width, 52pt min height). Touch target covers entire row.
Expand animation spring(duration: 0.3, bounce: 0.1) height from 52pt → 220pt. Content fades in with 0.08s delay.
Collapse animation easeOut(duration: 0.22) height to 52pt. Content fades out immediately (no delay).
Chevron rotation Collapsed: › (0°) → Expanded: ˅ (90°). Rotates with spring, same timing as height.
Cyan glow on-activate When any non-Solo mode selected: border fades to rgba(0,212,255,0.3) + top edge gradient appears.
Solo → Spread disabled In Solo mode, Spread slider animates to opacity 0.3 + isUserInteractionEnabled = false.
Default on load Collapsed. The Play tab scrolls to fit Hero + Pills + Brightness card without EdgeMixer taking space.
Persistence Expanded/collapsed state saved to UserDefaults. Survives app relaunch.
