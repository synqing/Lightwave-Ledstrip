Play Tab — Redesign Notes

One-Handed Thumb Zone

Hero LED preview (110pt) — compact, top of screen. Informational, not interactive. Beat pulse animation on every beat trigger.
Effect strip — horizontally scrollable chips at 80pt wide × 68pt tall. Swipe gesture on hero also cycles effects. All chips within thumb reach at ~320pt from top.
28pt tall sliders — generous target height. Touch anywhere on track adjusts value. Thumb indicator adds visual feedback without occupying more space.
EdgeMixer grid — 4-column grid, each cell 52pt min-height. 7 modes visible without scroll on standard screen.
Palette swatches — 56×56pt each. Bottom of scroll area, thumb-reachable. Gradient preview baked into swatch.

Motion Design Intent

Hero LED: scale(1.0 → 1.02) spring animation on beat, 150ms out / 200ms return.
BPM ring: stroke-dashoffset animates smoothly as confidence changes. Ring pulses scale(1.03) on beat.
Effect chip selection: border colour fade + glow shadow 180ms ease. Active chip scrolls into centre of strip.
EdgeMixer mode tap: background opacity flash (opacity 0.12 → 0.25 → 0.12) 200ms, communicates selection.
Slider drag: thumb glow shadow expands on touch-down (box-shadow 0 → 0 0 12px gold 0.6).

Haptic Feedback Points

Light Effect chip tap, palette swatch tap
Medium EdgeMixer mode change, slider endpoint (0 or max)
Heavy Swipe-to-change-effect gesture commit
Success Beat pulse every N beats (configurable, default: every beat)

10-Second First-Use Flow

0s: App opens, connected, LED preview visible immediately.
2s: User taps an effect chip — instant change, no modal required.
5s: User drags brightness slider — large target, obvious affordance.
8s: User swipes a palette swatch — colour change reflected in hero preview.
10s: User is in full control without ever opening a modal.
