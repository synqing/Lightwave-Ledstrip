Effect Browser — Redesign Notes

Layout Philosophy

68pt row height — WCAG 2.5.5 AAA target size (44pt minimum, 68pt preferred). Generous target for party-lighting conditions.
Animated LED preview thumbnail — 52×44pt canvas per row shows a live simulation of the effect colour pattern. Users see what they'll get before selecting.
Fixed bottom "Select" bar — no need to scroll to confirm. Current selection always visible. Thumb-reachable from any scroll position.
Category filter pills — top horizontal scroll. "Audio" gets a distinct gold-tinted pill to signal real-time reactivity category.

Search Behaviour

Filters live as user types — no submit button. Results update in 16ms (single frame) from JS-side filtering.
Matches on: effect name, category name. Fuzzy-tolerant (searches "lghp" → "LGP").
Empty state: magnifying glass icon + "No effects found" + "Try a different search" — never leaves user stranded.

Effect Row Anatomy

Preview
52×44pt animated canvas. Simulates colour motion at 30fps. Category-matched palette.
Tags
Category chip + Audio chip (gold, if audio-reactive) + New chip (for recent additions).
Selected
Gold border + background tint + checkmark. Scroll position preserved when closing/reopening.
Touch
Tap row → previews in bottom bar. Double-tap or "Select" button → commits and closes sheet.

Motion Design Intent

Sheet entrance: spring(damping:0.85) slide-up from bottom. 300ms. Backdrop blur fades in concurrently.
Category pill switch: selected pill slides + cross-fades. Row list reanimates with staggered fade-in (20ms per row, max 10 rows animated).
Row tap: scale(0.97) 80ms → scale(1.0) on release. Communicates physical press.
Bottom bar update: cross-fade 150ms as preview changes to selected effect.

Performance Notes

Preview canvases use off-screen OffscreenCanvas + requestAnimationFrame at 30fps (not 60fps) to halve GPU workload.
Only visible rows animate (IntersectionObserver gates canvas.animate()). Off-screen canvases are static.
SwiftUI implementation: LazyVStack with fixed row heights. TimelineView drives preview animation at 30fps independently of scroll.
