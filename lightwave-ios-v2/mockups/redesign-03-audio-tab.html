Audio Tab — Redesign Notes

Layout Hierarchy

Upper 55%: All real-time visualisations — spectrum canvas, octave bands, BPM, beat dot. These are read-only displays, so they live at the top where the user's eyes track without thumb interaction.
Lower 45%: Metric readouts, percussion triggers, DSP status. Scrollable if content exceeds space. All interactive elements thumb-reachable.
Vis mode selector — 4 modes (Spectrum, Waveform, Dot Matrix, Waterfall) in a segmented control. Matching the 4 existing canvas implementations in the codebase.

Visualisation Modes

Spectrum — 64-bin bar chart, logarithmic x-axis. Amplitude bars smoothed with exponential decay (fast rise, slow fall).
Waveform — rolling waveform line. Centre-referenced, gold stroke, translucent fill.
Dot Matrix — existing DotMatrixCanvas.swift adapted. 8×32 dot grid, intensity-mapped colour.
Waterfall — scrolling time-frequency plot. Each row is one spectrum frame. Colour = amplitude intensity (dark blue → gold → white).

Beat Dot Design

Dot pulses at exact BPM using esp_timer one-shot derived beat period. Animation duration matches the beat interval so it always lands "on time".
Colour: lwBeatAccent (#FFCC33) — slightly warmer than gold so it reads as distinct from other gold UI elements.
Glow radius expands from 8px to 28px on each pulse. Gives "lamp flashing" feel — reinforces the LED controller context.

Percussion Trigger Pills

Each pill flashes its border + background on trigger event. Onset = cyan, Kick = red, Snare = gold, Hi-Hat = green.
Onset flux sparkline below shows the last ~3 seconds of novelty curve. Threshold line drawn as dashed gold line. Peaks above threshold trigger onset events.

Performance Notes

Spectrum canvas uses TimelineView with .animation schedule — decoupled from SwiftUI redraw. Runs at 30fps max, independent of 120Hz ProMotion display refresh.
Octave band bars use withAnimation(.linear(duration: 0.08)) — fast attack, not spring, to avoid overshoot on transients.
Metrics grid values update at WebSocket frame rate (15-30 FPS). JetBrains Mono font for monospaced numerics prevents layout jitter.
