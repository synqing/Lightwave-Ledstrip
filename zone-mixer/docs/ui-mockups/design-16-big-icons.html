ZONE MIXER — DESIGN 16

Big Icons · 128×128 AtomS3 LCD · Pixel Art, No Text Labels

4× Scale (512×512)
1:1 Native (128×128)

Design Rationale

Philosophy

Text is slow to parse in dark, noisy venues. Large pixel-art icons communicate meaning instantly through shape and colour rather than language. Each icon is hand-drawn with fillRect calls — no image assets, no font rendering latency. The 32×32px icon size is optimised for the 0.85" LCD: large enough to read at arm's length, small enough to fit three per row.

Colour System

  • #FF4444 Zone 1 — warm red, highest visual weight.
  • #44FF44 Zone 2 — green, maximum contrast against Zone 1.
  • #4488FF Zone 3 — cool blue, completes the triad.
  • #FFB000 Amber accent — master bar, page dots, effect name.
  • #22CC44 / #CC2222 Status green/red — WiFi and WS connection state.

Page 1 — MIXER

Three 32×32px "LED block" icons represent zones. Each icon is a square outline in the zone colour, filled from the bottom proportionally to brightness — like a liquid level gauge. No numbers needed; the fill level IS the brightness. Below the three icons, a wide master bar spans the full width in amber.

Page 2 — WAVE

A large pixel-art waveform icon occupies the centre, representing the current effect visually. The effect name appears in small 10px text below — the only text on this page. Speed is shown as a directional arrow icon: slow (single arrow) to fast (triple arrow), with the arrow count reflecting speed range.

Page 3 — LINK

WiFi is shown as concentric pixel-art arcs (the universal WiFi symbol). WebSocket is shown as two opposing arrows. Both icons are green when connected, red when disconnected. The IP address appears in small text at the bottom — the only text on this page.

Live Overlay

When a parameter is being adjusted, a full-screen overlay appears showing: a contextual icon (sun for brightness, wave for speed), the value in large 48px pixel numerals, and a horizontal colour bar at the bottom. The overlay auto-dismisses after the user stops turning the encoder.

Accessibility

All icons use shape + colour redundancy — each zone icon has a distinct shape detail (single dot, double dot, triple dot) in addition to colour differentiation. Contrast ratios exceed 7:1 for all active elements against the #0a0a0a background.
